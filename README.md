
#K50全屏适配模块



### 写在前面
- 一个Magisk模块，直接刷入重启即可
- 从[release](https://github.com/Yejia995/K50FullScreen/releases/)页面下载zip包
- 帮助K50适配了横屏全屏，隐藏黑边
- 本模块在 嘟嘟ski 的 M11 全屏模块上魔改
- 不是转载，好用给赞，Yo

#

### 模块信息
- 模块作者 `Yejia995, 嘟嘟ski`
- 适用机型 `Redmi K50`
- 测试机型 `Redmi K50 Pro`
- 测试系统 `MIUI13（1307）`
- 兼容提示 `不支持其它机型`

#

#### 使用方式
- 在Magisk Manager中选择并刷入模块，重启手机即可！
